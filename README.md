#Generator
###These types of projects can be generated:
  - java (gradle)
    - spring-boot
    - quarkus
  - node js
  - go
---
###Requirements
**Install [node js](https://nodejs.org/en/)**

---
##Commands

Clean
`npm run clean`

Build
`npm run build`

Start application
`npm start`

Clean install and run
`npm run clean && npm build && npm start`
